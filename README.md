# BMO Desafio Frontend

Uma plataforma simples e intuitiva para que colaboradores possam criar, visualizar e gerenciar avisos internos.

## Tecnologias Utilizadas

- React
- Next.js
- TypeScript
- React context
- React Hook Form
- TailwindCSS
- Shadcn
- Zod

## Pré-requisitos

- npm
- Node v.20^

## Como rodar o projeto localmente

### 1 - Clone o repositório

```bash
  git clone https://github.com/julia-ctp/desafio-bmo-front.git
```

```bash
  cd desafio-bmo-front
```

### 2 - Instale as dependências

```bash
  npm i
```

### 3 - Rodar o projeto

```bash
  npm run dev
```
